# SEO 官方參考要點

Google Search Central 的 title link 文件指出，每個頁面都應有 `<title>`，標題應描述性且精簡，避免空泛、過長、關鍵字堆砌與重複模板化標題；也建議以簡潔方式在標題中加入品牌，並讓主視覺標題清楚對應頁面主題。

來源：https://developers.google.com/search/docs/appearance/title-link

Google Search Central 的 snippet/meta description 文件指出，Google 主要使用頁面內容產生摘要，但在 meta description 更能描述頁面時可能使用該內容。高品質 description 應提供簡短、相關、能吸引用戶的頁面摘要，且不同頁面應使用獨特描述，避免大量關鍵字堆砌。

來源：https://developers.google.com/search/docs/appearance/snippet
